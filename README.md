Simple Next.js App

This patch creates a minimal Next.js 13 app using the App Router.

Usage:
- Install: npm install
- Run dev: npm run dev

Files added:
- package.json
- app/layout.js
- app/page.js
- app/about/page.js
- app/globals.css

This is enough to run a tiny Next.js app that has a Home page and an About page.
